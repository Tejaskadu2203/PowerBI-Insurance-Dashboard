This project presents an end-to-end interactive Insurance Claims Analysis Dashboard built using Power BI.
It includes data transformation via Power Query, KPI generation using DAX, and business insights that help 
identify risk areas, customer claim trends, and policy profitability. The dashboard features dynamic filters,
segmented visuals, KPI blocks, and detailed claim performance tables to support data-driven decision-making.

Key Features:
✔ Data cleaning & transformation (Power Query)
✔ DAX-based KPIs (Premium Amount, Coverage Amount, Claim Amount, Delivery %)
✔ Claims by Age Group, Policy Type, and Status
✔ Active vs Inactive policy segmentation
✔ Trend analysis and business performance reporting
